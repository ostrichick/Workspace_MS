액션태그

jsp표준태그로 페이지 사이에서 이동을 제어하거나 자바빈을 생성할 때 주로 사용됨
<jsp:태그명 /> 문법으로 사용
jsp코드처럼 처리된 결과만 화면에 나타나므로 소스보기로 확인할 수 없다
xml 문법을 따른다 (xml: 다목적 마크업 언어, 데이터를 표현하거나 나타내는 문서형식)
반드시 종료태그 사용해야함
액션태그 사이에는 주석이 들어올 수 없다
액션태그의 속성값으로 <%= %> 를 사용할 수 있다

1. <jsp:include /> 
    jsp 지시자중 include와 동일한 역할
    - 포함되는 쪽header.jsp에서 선언한 변수를 포함한 쪽include.jsp에서 사용 불가능
    - 포함되는 쪽으로 request는 공유된다
2. <jsp:forward /> 
    - 현재 페이지로 들어온 요청을 다른페이지로 전달하는 기능
    - 포워드를 하게 되면 요청경로로 전달된 request가 그대로 다른 페이지로 전달되기때문에
      url은 유지되며 화면만 다른 페이지가 보이는 효과 (가상경로 구현)
   <jsp:useBean />, <jsp:setProperty />, <jsp:getProperty />
   - setProperty 태그는 자바빈즈의 속성을 대입할 때 사용하는 태그
     name 속성으로 빈즈를 선택하고 property 속성으로 대입할 필드 선택, value로 값 대입
     이 때 property 속성에 * 와일드카드 사용하면 파라미터 키와 일치하는 필드의 값을 자동대입
     또는 param속성을 사용해 필드와 파라미터 키가 다른 데이터도 대입 가능
   - getProperty 태그는 빈즈에서 특성 속성(필드)의 값을 꺼내옴
   
    - useBean 태그는 자바빈즈를 생성, 설정할때 사용
      * 자바빈즈: 데이터를 저장하기 위한 멤버변수(속성)과 게터/세터 메소드로만 이루어진 클래스
      * 자바빈즈 개발 규약: 
        - 자바빈즈는 기본패키지 이외의 패키지에 속해야한다
        - 멤버변수(속성) 접근제한자는 반드시 private
        - 기본생성자가 있어야함
        - public으로 된 게터/세터가 존재해야함
