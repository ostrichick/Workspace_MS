액션태그란?
-jsp 표준 태그로 페이지 사이에서 이동을 제어하거나 자바빈을 생성할 때 주로 사용됨
-<jsp:태그명 /> 문법으로 사용
-jsp 코드처럼 처리된 결과만 화면에 나타나므로 소스보기로 확인할 수 없다.
-xml 문법을 따른다. (xml:다목적 마크업 언어,데이터를 표현하거나 나타내는 문서형식)
-반드시 종료태그 사용한다.
-액션태그 사이에는 주석이 들어올수 없다.
-액션 태그의 속성값으로 <%=%> 를 사용할 수 있다.

1.<jsp:include /> - jsp 지시자 중 include와 동일한 역할을 한다.
                  - 사용할 때 주의할 점은 포함되어지는 쪽에서(header.jsp) 선언한 변수는
                    포함한 쪽(include.jsp)에서 사용할 수 없다.
                  - 포함되어 지는 쪽으로 request는 공유 된다. 
2.<jsp:forward/> - 포워드란 현재 페이지로 들어온 요청을 다른 페이지로 전달하는 기능
				 - 포워드를 하게 되면 요청 경로로 전달된 request가 그대로 다른 페이지로
				   전달되기 때문에 url은 유지되며 화면만 다른 페이지가 보이는 효과를 얻을 수
				   있다. (주로, 가상경로 구현시 사용)
3.<jsp:useBean/>,<jsp:setProperty/>,<jsp:getProperty/>
  -useBean 태그는 자바빈즈를 생성하거나 설정할 때 사용.
  -setProperty 태그는 자바빈즈의 속성을 대입할 때 사용하는 태그이다.
   name속성으로 빈즈를 선택하고 property 속성으로 대입할 필드를 선택하고  value 속성으로 값을
   대입한다.
   이때 property 속성에 *(와일드카드)기호를 사용하면 파라미터 키와 일치하는 필드들에대하여 값을
   자동 대입할 수 있다.
   또는 param 속성을 사용하면 필드와 파라미터 키가 다른 데이터도 대입할 수 있다.
  -getPropery 태그는 빈즈에서 특정 속성(필드)의 값을 꺼내온다.
  
  *자바빈즈란?
  -데이터를 저장하기 위한 멤버변수(속성) 과 게터/세터 메서드로만 이루어진 클래스를 의미한다.
  *자바빈즈 개발 규약
  -자바빈즈는 기본 패키지 이외의 패키지에 속해야 한다.
  -멤버 변수(속성)의 접근제한자는 private로 선언되어야 한다.
  -기본생성자가 있어야 한다. 
  -게터/세터가 존재해야한다.
  -게터/세터는 반드시 접근제한자를 public으로 선언해야한다.
